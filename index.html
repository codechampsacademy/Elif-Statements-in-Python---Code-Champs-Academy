elif في بايثون

في بايثون، تستخدم elif (اختصار لـ "else if") لإضافة شرط جديد بعد التحقق من شرط if. إذا كان الشرط الأول غير صحيح، يتم التحقق من شرط elif. يمكن استخدام العديد من العبارات elif في البرنامج.

مثال 1: تحديد اليوم بناءً على الرقم
number = int(input("Enter a number (1-7): "))
if number == 1:
 print("Today is Monday")
elif number == 2:
 print("Today is Tuesday")
elif number == 3:
 print("Today is Wednesday")
elif number == 4:
 print("Today is Thursday")
elif number == 5:
 print("Today is Friday")
elif number == 6:
 print("Today is Saturday")
elif number == 7:
 print("Today is Sunday")
else:
 print("Invalid number. Please enter a number between 1 and 7.")

في هذا المثال، يطلب البرنامج من المستخدم إدخال رقم يمثل اليوم. يتم التحقق من الرقم باستخدام سلسلة من عبارات if وelif. إذا لم يكن الرقم بين 1 و7، يتم طباعة رسالة خطأ باستخدام else.

مثال 2: مقارنة الدرجات
grade = int(input("Enter your grade: "))
if grade >= 90:
 print("Excellent!")
elif grade >= 80:
 print("Great job!")
elif grade >= 70:
 print("Good effort!")
elif grade >= 60:
 print("You passed.")
else:
 print("Unfortunately, you failed. Try harder next time.")

هنا، يقوم البرنامج بمقارنة الدرجة التي يدخلها المستخدم مع قيم معينة. بناءً على الدرجة، يتم طباعة رسالة ملائمة. إذا كانت الدرجة أقل من 60، يتم عرض رسالة تحفيزية.

ملخص

تُستخدم elif عندما يكون هناك أكثر من شرط نحتاج للتحقق منه. يتيح لنا هذا تجنب كتابة شروط متعددة باستخدام if فقط، مما يجعل الكود أكثر تنظيمًا وأسهل قراءة.
